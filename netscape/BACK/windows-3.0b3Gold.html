ATLAS GOLD PREVIEW 2 FOR WINDOWS

    You are using Atlas Gold Preview 2 (Beta) for Windows 95. This product includes all the functionality of Atlas Preview 2, plus the Gold features that enable you to edit pages visually.

    These release notes cover:

    • Requirements
    • Editor Known Problems
    • Editor Features

    For information on browser-specific features and bugs, see http://home.netscape.com/eng/mozilla/3.0/relnotes/windows3-0b3.html

    This software contains code that disables most of its features on July 31, 1996. You will be reminded of this starting July 15, 1996.

REQUIREMENTS

    To run the 32bit Navigator, you MUST have a 32-bit TCP/IP stack. Both Windows 95 and Windows NT provide 32bit TCP/IP stacks that you can setup. If you need help setting up the TCP/IP stack in Windows 95, see http://www.windows95.com.

EDITOR KNOWN PROBLEMS

    NOTE: This release includes the newly-added tables feature. If you plan to use this feature, please be aware that it is still under development and therefore you should save your work often!

    Tables

    • Table navigation: you cannot use the Up and Down arrow keys to move the cursor within a table. Instead, use the Left and Right arrow keys or the mouse to move the cursor from cell to cell.
    • Online help is not yet available for the table dialog boxes.
    • Inserting a table: if you insert a table while you have something selected (such as a block of text) at the end of a document, the application will crash.
    • Multiple tables: you cannot insert lines of text between adjacent tables. Use this workaround: choose Edit Document Source from the View menu to manually edit the HTML tags.
      1. For each table, locate the table boundary tags </table> <table> (where one table ends and the next table begins).
      2. Insert the paragraph tag <p> between the table tags.
      3. Type some text and then insert the closing paragraph tag </p>.

        4. For example:

        </table>

        <p>some text here</p>

        <table>

    • Tables at the beginning of documents: you cannot insert text before a table at the beginning of a document. Use this workaround: choose Edit Document Source from the View menu to manually edit the HTML tags:
      1. Locate the beginning table tag <table>.
      2. Insert the paragraph tag <p> before the table tag.
      3. Type some text and then insert the closing paragraph tag </p>.

        4. For example:

        <p>some text here</p>

        <table>

    • Tables at the end of documents: you cannot insert text after a table at the end of a document. Use this workaround: choose Edit Document Source from the View menu to manually edit the HTML tags:
      1. Locate the closing table tag </table>.
      2. Insert the paragraph tag <p> after the table tag.
      3. Type some text and then insert the closing paragraph tag </p>.

        4. For example:

        </table>

        <p>some text here</p>

    • Undo does not function correctly for removing a table after you've inserted it.
    • Deleting tables: you can delete any table except one that is at the end of a document.
    • Table cell properties: you cannot set properties for multiple cells by selecting them and then using the Cell|Properties menu item. You have to select the cells one at a time. You can set properties for all cells in a selected row by using the Row|Properties menu item.

    Other known problems:

    • Editing a remote page that contains an animated .GIF file will cause the application to crash.
    • Printing:
      • Cannot cancel print during formatting stage (on print progress dialog). Application will hang.
    • Images:
      • Inserting images: When you click the Original Size button in the Image Properties dialog box, the dimensions display as 0 (zero). This is temporary and does not affect the original size of the image. Click OK or Apply to exit the dialog box. Select the image you inserted and then display the Image Properties dialog box again to view the correct dimensions.
      • Changes made by resizing an image through an external image editor aren't automatically reflected in the editor. Use the Image Properties dialog box to adjust the dimensions as well.
    • On some systems, once you change the font size in the Options|General|Fonts preference panel you cannot change it back. Workaround: choose another font, and then reset the font and font size.
    • If you have Plug-ins on your regular installation of the Netscape Navigator, you must repeat the installation procedure for Navigator Gold (unless you install Navigator Gold in the same directory)
    • If you edit a document that has invalid HTML, it may appear differently in the browser than in the editor.
    • International character support: 
      • Although doublebyte input is supported, inline input is not.
      • On multibyte systems, buttons in some dialog boxes will display incorrectly. For example, if you install Navigator Gold on a machine running Japanese Win95 or NT 3.51, the Cancel, Apply, and Help buttons appear as garbage characters.

EDITOR FEATURES

  • Tables feature: Select Table from the Insert menu and use the Create Table dialog box to specify the number of columns and rows, and other attributes for the table, such as height, width, cell color, and caption placement.
    • Once you've created the table, you can add additional rows, columns, and cells by using the Row, Column, and Cell items on the Insert menu.
    • Use the Table, Row, and Cell items on the Properties menu to change the attributes for the currently selected table, row, or cell.
  • FTP URLs can now specify a relative path instead of specifying a path from the root directory. This is especially useful when specifying an upload directory. For example, suppose you have a directory called public_html in which you store your html files. Normally using the browser, you would have to specify the absolute path ftp://lloyd@foo.isv.com/public/lloyd/public_html. Instead, you can specify a relative path with ftp://lloyd@foo.isv.com/./public_html. The /./ replaces /public/lloyd/ in the URL. This is important because over time, your absolute path may change, but the relative path will always remain the same.
  • Shortcut keys:
    • To enter one or more non-breaking spaces, press Shift+Spacebar.
    • To enter a line break, press Shift+Enter.
  • Use the HTML Tag feature to add Java applets and Netscape plug-ins. Choose HTML Tag from the Insert menu to display the HTML Tag dialog box.
