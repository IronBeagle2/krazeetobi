the list server. Sending each part in the correct numerical order does
not guarantee each part will arrive in the correct numerical order. You
can increase your chances by waiting 3 minutes between sending each
part. A delay of several hours between sending parts is almost a guarantee,
albeit inconvenient. BTW, the same rule applies for uploading large
files to alt.binaries.* newsgroups.
