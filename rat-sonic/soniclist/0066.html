Majordomo is the name of the software that manages the list. Ratman and
I are co-owners of the list.

I usually use pine for mail, and I've not had that problem. The list
messages arrive with both "From:" and "Reply-To:" fields in the headers.
When I use "r", pine asks if I want to use Reply-To instead of From. The
Reply-To field is set to the list and the From field is set to the
message's writer. Elm should be configurable to work that way, but never
having used it, I can't tell you what you need to do.

I'll review the intro file to see about moving the address for posting,
and maybe make a few more tweaks at the same time.
