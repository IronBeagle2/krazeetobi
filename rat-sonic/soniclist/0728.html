Archives of the Sonic mailing list for February and March are now
available for ftp. The filenames are soni0295.zip and soni0395.zip,
and can be found on ftp.mv.com at /pub/users/rat/sonic/message-archive.
Don't forget to use Binary or Image transfer mode. Each .zip is
about 450K and expands to about 1100K.

Archives of the current month (April) can, as always, be retrieved
through the mailing list software. See the Sonic "How-To" file for
instructions.

If you use a Unix system, you can load all of the archive files as a
mailbox folder for convenient browsing with pine or elm.
