The Sonic mailing list is now on the Web. There's still some work to be
done on it, but all the links seem to be working and it's more or less
ready for prime time.

The URL is:
ftp://hff.mv.com/pub/sonic/mobius.html

That is ftp:// (don't use http).

The Web page does not replace the ftp site (hff.mv.com). In fact, not
all of the files on the ftp site are represented in this page. This is
just another way to use the ftp site.

Where are the graphics, sounds, and all the other cool stuff that makes
the Web fun? The ftp'able files are all in their usual places. But
there are no pictures or sounds on this page. I did include a .GIF image
earlier, but my browser choked on it. I've not confirmed this, but it
may be because the server does not support http.

If you know of any Sonic-related Web pages not included, please send me
the URL so I can add them as links.
